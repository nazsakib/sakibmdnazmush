Welcome to the **Terminal Matrix Portfolio**—not just a static resume, but a fully functional, gamified terminal emulator running entirely in the browser. Built to showcase technical skills through an unforgettable "Red Pill / Blue Pill" experience.

---

## ✨ Features

### 🖥️ Immersive Terminal Interface
- **Red Pill / Blue Pill Intro:** A cinematic typing sequence that presents visitors with a choice. Taking the Red Pill triggers an authentic Linux `dmesg` style boot sequence.
- **Smart CLI Emulator:** Supports standard terminal UX, including `Tab` autocompletion and `Up`/`Down` arrow command history.
- **Typo Correction Engine:** Implements a Levenshtein Distance algorithm to detect mistyped commands (e.g., typing `hepl` prompts: *"Did you mean 'help'?"*).

### 🎨 Next-Level Visuals & Theming
- **Dynamic Theming:** Change the terminal's color palette on the fly with the `theme` command (supports Hacker, Cyberpunk, and Dracula).
- **Advanced CRT Effects:** CSS-powered scanlines, radial vignette curvature, text glow, and random digital glitch animations.
- **Matrix Digital Rain:** An optimized HTML5 Canvas background that drops classic Katakana characters.

### 🕹️ Gamification & Easter Eggs
- **Playable ASCII Snake:** Type `play snake` to launch a fully playable, keyboard-controlled version of Snake rendered entirely in ASCII art right inside the terminal.
- **Konami Code:** Entering the classic `Up, Up, Down, Down, Left, Right, Left, Right, B, A` sequence transforms the Matrix code rain into falling emojis.
- **System Hack:** The hidden `hack` command triggers a chaotic system failure sequence and reboot.

### 🔌 Live Data & Smart Integrations
- **AI Simulator (`ask`):** A simulated AI assistant that answers conversational queries about my location, stack, and availability.
- **Live GitHub Stats:** The `github` command fetches and renders real-time data directly from the GitHub API.
- **System Status:** The `status` command pulls the user's actual local browser time and User-Agent data.

### 📱 Superior UX Engineering
- **Procedural Synthesized Audio:** Instead of overlapping audio files, every keystroke is dynamically synthesized using the **Web Audio API** with randomized pitches for true mechanical keyboard realism.
- **Mobile Hacker Keyboard:** On mobile devices, the native keyboard is bypassed in favor of a sleek, custom on-screen hacker keyboard, preventing layout-breaking zoom issues.
- **State Persistence:** Uses `localStorage` to remember returning visitors, automatically skipping the intro and welcoming them back.

## ⌨️ Commands

| Command | Description |
| :--- | :--- |
| `help` | Displays the list of available commands |
| `about` | Prints a summary of my background and experience |
| `projects` | Lists my latest portfolio projects |
| `skills` | Renders dynamic ASCII progress bars of my technical skills |
| `github` | Fetches live GitHub API statistics |
| `status` | Prints local system connection data and time |
| `contact` | Displays my email, GitHub, and LinkedIn links |
| `message` | Automatically opens your default email client |
| `resume` | Initiates a download of my PDF resume |
| `theme <name>`| Switches the color scheme (`hacker`, `cyberpunk`, `dracula`) |
| `play snake` | Launches the interactive ASCII Snake game |
| `ask <query>` | Asks the terminal a conversational question |
| `mute/unmute` | Toggles the mechanical keyboard sound effects |
| `clear` | Clears the terminal screen output |
| `exit` | Terminates the session and closes the tab |
